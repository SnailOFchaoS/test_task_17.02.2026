# test_task_17.02.2026

Для работы у вас должен быть установлен Node.js (я использовал v22.14.0) и пакетный менеджер npm или yarn (я использовал npm 10.9.2).

**Win10–11**

1. Клонируйте репозиторий себе на машину.
2. Откройте командную строку и перейдите в корневую папку клонированного репозитория.
3. Установите все необходимые зависимости командой `npm install`.
4. Запустите приложение командой `npm run dev`.
5. Откройте приложение в браузере по адресу `http://localhost:3000/`. Если порт 3000 занят, в консоли при сборке будет указан фактический порт — откройте его.

Если нужен только результат без запуска локально:
